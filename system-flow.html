เกิดอะไรขึ้นเบื้องหลัง

  1. พี่พิมพ์ "ยินดี" → บอทเช็คว่าเป็นเจ้าของ/ทีม → ส่งเมนู
  2. กด 📅 นัดประชุม → เปิด LIFF (mini app) พร้อมเลือกลูกค้าของกลุ่มนั้นให้
  3. กรอกฟอร์ม → ระบบสร้าง Calendar event + Google Meet ในปฏิทิน moodata.me
  4. ผู้เข้าร่วมได้ อีเมลเชิญ (invite) เด้งเข้าปฏิทินอัตโนมัติ
  5. บอท push การ์ดนัด + Meet link เข้ากลุ่มลูกค้าทันที
🔒 ปลอดภัย: เฉพาะพี่กี้ + ทีมใน whitelist เท่านั้นที่จองได้ · ลูกค้าในกลุ่มกดสร้างนัดเองไม่ได้
