คู่มือ 05 · จัดการงาน

เว็บแอปจัดการงานลูกค้า

เปิดจากปุ่ม 🖥️ บนแถบเมนู
✓ ใช้งานได้

หน้าเว็บแอปมี 3 แท็บหลัก:

  1. งานของฉันงานที่ได้รับมอบหมาย · กด 📤 ส่งตรวจ เมื่อเสร็จ · 🔄 โอนงาน ให้เพื่อนได้
  2. งานทั้งหมดเห็นงานทุกบริษัท · เจองานว่างที่อยากช่วย กด 🙋 รับงาน ได้เลย
  3. Projectดูงานแบ่งตามลูกค้า — 1 กลุ่มไลน์ = 1 Project · เลือก Project ที่ช่องด้านบนแล้วเห็น % ความคืบหน้า + งานแยกเป็น 6 สถานะ (ใหม่ / กำลังทำ / รอตรวจ / เสร็จ / เกินกำหนด / ยกเลิก)

อยากเปิดงานใหม่กด ➕ เปิดงานใหม่ → เลือกประเภทงาน / ลูกค้า / รายละเอียด / มอบหมาย / กำหนดส่ง

งานแยกเป็นประเภท — มีสี + ไอคอนให้เห็นชัด ทั้งในเว็บและการ์ดแจ้งเตือนใน LINE:
